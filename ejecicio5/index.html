En HTML, los elementos se pueden clasificar como "en bloque" o "en línea", lo que determina cómo se renderizan en la página web. Los elementos de bloque ocupan todo el ancho disponible y siempre comienzan una nueva línea, mientras que los elementos en línea solo ocupan el espacio necesario para su contenido y pueden estar en la misma línea que otros elementos
En este ejemplo, el párrafo "p" será un elemento de bloque, mientras que el "span" será un elemento en línea.

Elementos de bloque se utilizan para la estructura principal de la página; elementos en línea se utilizan para el formato de contenido dentro de esa estructura.
