Futbol

En el fútbol, el sistema de puntuación es sencillo: una victoria otorga 3 puntos, un empate 1 punto y una derrota 0 puntos. La diferencia de goles (goles marcados menos goles recibidos) se usa como desempate. El sistema de puntuación en el fútbol: Victoria: 3 puntos para el equipo ganador. Empate: 1 punto para cada equipo. Derrota: 0 puntos para el equipo perdedor.

El objetivo principal para ganar en fútbol es marcar más goles que el equipo contrario durante el tiempo reglamentario del partido. Esto se logra llevando el balón hacia la portería contraria y, finalmente, introduciéndolo en ella sin usar las manos ni los brazos, salvo en el caso del portero.
