Que son las medidas de css

Las unidades de medida en CSS son los tipos de datos que se usan para definir el tamaño de las propiedades en CSS, como el ancho, la altura, el margen, el relleno, el tamaño de fuente, etc. Existen dos grandes categorías de unidades de medida: absolutas y relativas.

Unidades de medida CSS absolutas

Las unidades de medida absolutas se utilizan para definir tamaños fijos en los elementos de una página. Estos tamaños pueden expresarse mediante medidas digitales, como los píxeles (px), o mediante medidas físicas, como los centímetros (cm), milímetros (mm), etc.

Unidades de medida CSS relativas

Las unidades de medida relativas se utilizan para definir tamaños dinámicos en los elementos de una página. Estos tamaños pueden expresarse en relación a diferentes elementos como la resolución de pantalla, el tamaño de la ventana del navegador, el tamaño de la tipografía utilizada, etc.
